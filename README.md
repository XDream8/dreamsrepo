# dreamsrepo
My Repository for Gentoo

## Add this repository
```sh
# eselect repository add dreamsrepo git https://github.com/XDream8/dreamsrepo.git
```
## Sync
**Using emaint**
```sh
# emaint sync -r dreamsrepo
```
